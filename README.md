💻 Petulap - Control de Producción v6.3
Petulap es una solución integral de gestión y control de producción diseñada para optimizar el flujo de trabajo en plantas de reacondicionamiento tecnológico. El sistema facilita la comunicación en tiempo real entre las áreas de Ventas, Administración y Planta Técnica, asegurando un seguimiento preciso de cada equipo procesado.

🚀 Funcionalidades Principales (v6.3)
🛠️ Gestión de Lotes en 4 Fases
La versión 6.3 introduce un sistema de producción granular que divide el trabajo en etapas críticas:

Clonación: Registro inicial y carga de sistemas operativos.

Testeo: Pruebas de hardware detalladas.

Limpieza: Reacondicionamiento físico y estético.

Encaje: Empaque final y preparación para despacho.

📊 Panel Maestro & Analytics
Los administradores cuentan con herramientas de supervisión avanzada:

Dashboard en Tiempo Real: Visualización de técnicos activos, tareas actuales y tiempos de ejecución.

Métricas de Éxito: Gráficos dinámicos (SVG) que muestran la tasa de éxito diaria, equipos en soporte y promedios de productividad.

Exportación de Datos: Generación de reportes detallados en formatos CSV y PDF para auditorías semanales.

🎨 Experiencia de Usuario & Accesibilidad
El sistema adapta su interfaz a las necesidades del usuario:

Temas Personalizados: Incluye modos Dark (estilo Discord), Minimalista Moderno (juvenil) y Professional 50+ (optimizado para público senior con mayor legibilidad).

Feedback Inmersivo: Motor de audio de 8 bits para confirmaciones sonoras y notificaciones visuales con animaciones fluidas.

Reloj Digital: Navbar con cronómetro de matriz de puntos para control de jornada.

🚨 Sistema de Notificaciones & Prioridades
Gestión inteligente de urgencias:

Alertas de Emergencia: Notificaciones intrusivas para tareas críticas que requieren atención inmediata.

Cuentas Regresivas: Monitor de tiempos límite para lotes con prioridades "Urgente" o "Urgentísimo".

🛠️ Stack Tecnológico
Frontend: HTML5, CSS3 (Tailwind CSS) y JavaScript (Vanilla JS).

Backend: Supabase (PostgreSQL + Real-time).

Iconografía: FontAwesome 6.4.0.

Audio: Web Audio API (Motor 8-bits personalizado).

📂 Estructura del Repositorio
web/index_mejorado.html: Versión actual con sistema de lotes, temas avanzados y analytics.

web/spa ORIGINAL/MASTER SPA.html: Base histórica del sistema Single Page Application (v6.2).

web/js/config.js: Configuración global de Supabase, roles y constantes del sistema.

web/js/app.js: Lógica principal de la aplicación, manejo de estados y eventos.

👥 Roles del Sistema
Técnico: Registro de producción, gestión de fases de lotes y recepción de misiones.

Ventas (André): Emisión de requerimientos logísticos, radar de despacho y confirmación de entregas.

Admin / Owner: Supervisión global, reasignación de tareas, auditoría de lotes y exportación de reportes.
